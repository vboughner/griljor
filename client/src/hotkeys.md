# Griljor Hotkeys

## Movement

Movement is 8-directional. The QWERTY layout mirrors the original 1989 game:

```
Q  W  E      NW  N  NE
A     D       W     E
Z  X  C      SW  S  SE
```

| Key(s) | Direction |
|---|---|
| `W` / `↑` / Numpad `8` | North |
| `X` / `↓` / Numpad `2` | South |
| `D` / `→` / Numpad `6` | East |
| `A` / `←` / Numpad `4` | West |
| `Q` / Numpad `7` | Northwest |
| `E` / Numpad `9` | Northeast |
| `Z` / Numpad `1` | Southwest |
| `C` / Numpad `3` | Southeast |

## Items

| Key | Action |
|---|---|
| `S` (lowercase) | Pick up item at your feet → left hand |
| `Shift+S` | Pick up item at your feet → right hand |
| `Shift+Z` | Drop left-hand item |
| `Shift+X` | Drop right-hand item |

Note: lowercase `z` and `x` are movement (SW/S); the uppercase `Shift` versions are drop actions.

## Chat

| Key | Action |
|---|---|
| `T` | Focus the chat input |
| `Enter` | Send chat message |
| `Escape` | Exit chat, return focus to game |

## Mouse

| Action | Result |
|---|---|
| Left-click tile | Fire right-hand weapon toward tile (or pick up item) |
| Middle-click tile | Fire left-hand weapon toward tile (or pick up item) |
| Right-click tile | Step one tile toward clicked position |
